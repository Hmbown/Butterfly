How to share weights between models:

GAP 4: Active-batched (K4) path with circular=True
---------------------------------------------------

Minimal setup:
  from bna.mlx.attention import wayfinder_permute_window_attention_active_batched
  B, Hq, Hkv, dh, T, W = 1, 2, 2, 32, 64, 8
  Tq = 4

  k = mx.array(rng.standard_normal((B, Hkv, T, dh)).astype(np.float32))
  v = mx.array(rng.standard_normal((B, Hkv, T, dh)).astype(np.float32))
  query_positions = mx.arange(T - Tq, T, dtype=mx.int32)

  # shape [Hq, 1, T] for single cycle

The naive reference for circular: for each query at permuted position p, gather K/V from
circular window [(p-W)%T ... (p+W)%T], apply causal mask (original index j <= i), softmax, matmul.
